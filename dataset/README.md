# Dataset

## Source

**SKU-110K** via Roboflow Universe
- URL: https://universe.roboflow.com/jacobs-workspace/sku-110k
- ~11,000 images of densely packed retail shelves
- 1 class: `objects` (individual product instances)

## Download

```bash
pip install roboflow

python dataset/download.py
```

## Structure (after download)

```
dataset/
├── data.yaml          # Auto-generated by Roboflow (class names + paths)
├── train/
│   ├── images/
│   └── labels/
├── valid/
│   ├── images/
│   └── labels/
└── test/
    ├── images/
    └── labels/
```

> **Note:** Image files are `.gitignore`d. Download locally or from S3.
